# AI Interview Coach｜AI 面试训练 Demo

## 在线体验

https://ai-interview-demo-tau.vercel.app

## 1. 项目简介

AI Interview Coach 是一个 AI 面试训练产品 Demo。

用户可以选择目标岗位，系统根据岗位生成对应面试问题。用户输入回答后，系统会根据回答内容生成评分与改进建议。

这个 Demo 用于展示 AI 产品经理 / AI 产品助理岗位所需要的产品理解能力、快速原型能力和 Vibe Coding 实践能力。

---

## 2. 目标用户

主要面向正在准备面试的求职者，尤其是：

* 应届生
* 转行求职者
* 产品经理候选人
* AI 产品助理候选人
* 留学顾问、运营等岗位候选人

---

## 3. 用户痛点

传统面试准备存在几个问题：

* 不知道面试官会问什么
* 回答没有结构
* 缺少即时反馈
* 不知道自己的回答哪里需要改进
* 找真人模拟面试成本较高


---

## 4. 核心功能

### 岗位选择

用户可以选择不同岗位，例如：

- 产品经理
- AI 产品助理
- 留学顾问
- 运营岗位

### 面试题生成

系统根据岗位展示对应的高频面试问题。

### 回答输入

用户输入自己的面试回答。

### 评分反馈

系统根据回答内容给出：

- 综合评分
- 表达建议
- 结构优化建议
- 岗位匹配建议

---

## 5. 产品价值

这个产品可以帮助用户：

1. 快速进入面试训练状态
2. 提升表达结构
3. 发现回答中的问题
4. 降低模拟面试成本
5. 提高面试准备效率

---

## 6. 技术实现

本项目使用：

- Next.js
- React
- TypeScript
- Tailwind CSS
- Cursor
- Vibe Coding 工作流

当前版本是前端交互 Demo，后续可以接入 OpenAI API，实现真正的 AI 评分与反馈。
